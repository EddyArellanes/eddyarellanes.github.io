Creando el Sitio

¿Como construí ésta web?

Como Developer, soy muy curioso siempre que visito un sitio me gusta explorar en qué están hechos y con qué herramientas.

Es por ello que hoy les comparto sin que usen Wappalyzer o BuiltWith, como construí mi sitio y qué se viene después, entonces empecemos


Todas las Páginas Web, usan por regla HTML5, Css3 y Javascript, es decir js se puede desactivar en los navegadores, pero hoy en día son el corazón de las webs entonces nadie incurriría a desactivarlo en estos días.

Tecnologías Usadas en la Página de Eddy Arellanes Bastida 2018 Beta

(Sí, ésta es la versión Beta de la Página, espero poder liberar la versión final)

  • HTML5
  • CSS3
  • MaterializeCSS Alpha 1.0 (Es Fantástica)
  • Github Pages

Para ésta primera versión sólo utilice Javascript para inicializar una función de MaterializeCss, por ende no considero que lo haya usado y no lo puse.

Algunos estilos personalizados de espaciado y formato a textos se los dí con Css3, y por último maqueté la Página con HTML usando tags de HTML5 como los header, main, footer, nav, articles, etc.

¿Qué sigue?

Fase 1: Bueno, primero que nada quiero terminar de desarrollar la sección de Portafolio, Optimizaré SEO en las páginas ya existentes.

Fase 2: Implementar React para automatizar mi Página volviéndola Componentes, esto es una ventaja para darle mantenimiento a una Página

Fase 3: Añadir un Backend

Para administrar la información de la web como el Blog, para esto estoy aún contemplando entre dos opciones:

  • Python + Django
  • Firebase

Es importante elegir bien porque en base a eso se decidirá el futuro de mi sitio

Fase 4: Añadir Server Side Rendering,

como sabemos React renderiza todo desde el lado del cliente, por ello quiero una vez implementado el Backend, crear el SSR para poder renderizar la información desde el principio y los Robots puedan encontrar mi Data

Fase 5: Última FaseMigración a IaaS o PaasS

Github es cool y le tengo mucho cariño, sin embargo para futuras cosas, necesitaré tener el control del servidor donde alojo mi sitio, por lo que planeo migrar este sitio a una de éstas opciones:

  • Amazon Web Services
  • Microsoft Azure
  • Digital Ocean

De momento están ordenadas según me convencen más, tengo preferencia por Amazon por la sencilla razón que tengo más experiencia con él, con Azure tengo muy poca, y con Digital Ocean nada.

Adicional: Metodología de Proyecto

Planeo aplicar una metodología de trabajo PMP pero agregando Metodologías Ágiles, estoy entre Scrum y EXtreme Programming(XP), espero aplicarlas con Disciplina o no servirán de nada.
